# Integrating Sentiment Analysis of Business News Headlines from *The Manila Times* for Predicting the Philippine Stock Exchange Index (PSEi)

## Description
This repository contains the coursework project entitled **“Integrating Sentiment Analysis of Business News Headlines from *The Manila Times* for Predicting the Philippine Stock Exchange Index (PSEi)”**. The study examines the relationship between sentiment derived from business news headlines and movements in the Philippine Stock Exchange Index (PSEi) using natural language processing and statistical modeling techniques.
